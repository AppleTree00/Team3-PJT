## 1. 문제 정의

### 취업 준비생이 겪는 3가지 핵심 고통

> **"이력서를 잘 쓰고 싶은데, 어떻게 고쳐야 할지 모르겠다."**

| 문제 | 현실 |
|------|------|
| 이력서 품질 불확실 | 직무 키워드·성과 중심 작성법을 모르는 경우가 대부분 |
| 피드백 접근성 부재 | 전문 커리어 코치 비용 高, 지인 피드백 한계 |
| 지원 현황 관리 어려움 | 여러 회사·포지션 지원 상태를 스프레드시트로 수동 관리 |

---

## 2. 솔루션 소개

### a4u — AI가 함께 완성하는 나의 이력서

**a4u**는 AI 기반 이력서 코칭과 취업 지원 현황 관리를 하나의 플랫폼으로 제공합니다.

- **AI 코칭:** GPT-4o / Claude 기반 실시간 이력서 피드백
- **이력서 빌더:** 섹션별 구조화된 이력서 작성 + 미리보기
- **지원 현황 트래킹:** 전체 지원 현황을 대시보드에서 한눈에
- **템플릿 제공:** 직군별(IT·경영·일반) 최적화 이력서 템플릿

---

## 3. 핵심 기능

### 3-1. AI 이력서 코칭

- **Before / After 방식** 피드백 제공
- 직군 카테고리별 특화 분석 (IT / 경영·관리 / 일반)
- 강점·개선점·핵심 키워드 3종 리포트
- 실제 GPT-4o-mini, Claude-3-Haiku 연동 (데모 fallback 지원)

### 3-2. 이력서 빌더

- 인적사항 / 전문가 요약 / 경력 / 학력 / 스킬 섹션 구조화
- 파일 업로드 지원 (PDF, DOCX)
- 실시간 미리보기 패널
- 작성 상태 자동 저장 (localStorage)

### 3-3. 취업 지원 관리 대시보드

- 지원 현황 단계별 관리: `초안 → 제출 → 검토중 → 최종합격 / 불합격`
- 지원 이력 통계 요약 카드
- 메모 기능으로 면접 일정·피드백 기록

### 3-4. 관리자 콘솔

- 사용자 현황 모니터링
- 이력서 템플릿 업로드·관리 (PDF/WORD)
- AI 자연어 SQL 마이그레이션 콘솔

## 4. 서비스 흐름 (User Journey)

```
[회원가입 / 로그인]
       ↓
[이력서 작성 방법 선택]
  ├─ 직접 작성 (Builder)
  ├─ 파일 업로드 (PDF/DOCX)
  └─ 템플릿 선택
       ↓
[이력서 완성]
       ↓
[AI 코칭 받기]  ←→  [Before / After 피드백 확인]
       ↓
[지원서 제출 & 현황 트래킹]
       ↓
[대시보드에서 전체 지원 현황 관리]
```

## 5. 기술 스택

| 구분 | 기술 |
|------|------|
| **Frontend** | HTML5 / Vanilla JS / Tailwind CSS |
| **Backend** | Python Flask / Flask-SQLAlchemy |
| **Database** | SQLite (JSON 컬럼 확장) |
| **AI** | OpenAI GPT-4o-mini, Anthropic Claude-3-Haiku |
| **인증** | 세션 기반 인증 / 프로필 이미지 업로드 |
| **배포** | Gunicorn / Procfile (PaaS 호환) |

> **3일 MVP** 목표로 설계된 경량 아키텍처 — 빠른 검증과 반복에 최적화

## 8. 현재 개발 현황

### 완료된 기능 ✅

- 회원가입 / 로그인 / 비밀번호 재설정
- 프로필 관리 (이름·이메일 수정, 비밀번호 변경, 아바타 업로드)

- 취업 지원 현황 대시보드

## 9. 로드맵

```
Phase 1 (현재) — MVP 완성
  ✅ 핵심 기능 구현 및 안정화

Phase 2 (1~2개월)
  □ 이력서 PDF 내보내기
  □ 다중 이력서 버전 관리
  □ JD 붙여넣기 → 맞춤 이력서 추천

Phase 3 (3~6개월)
  □ 모바일 앱 (React Native / Expo)
  □ 채용공고 연동 (스크래핑 또는 API)
  □ 커리어 코치 매칭 마켓플레이스

Phase 4 (6개월~)
  □ 기업 HR 도구 연동 (ATS 호환)
  □ B2B 기관 제휴 확대
```
